# Abp.CmsKit

## Demo plugin using Abp.CmsKit

Demo project for Abp.CmsKit, which is a plugin for ABP Framework.

### Pre-requirements

Build the project by using the following command:

```bash
dotnet publish -c Release
```

Pack the dlls to a NuGet package by using the following command:

```bash
 dotnet pack -c Release -p:NuspecFile=Abp.CmsKit.nuspec
```
